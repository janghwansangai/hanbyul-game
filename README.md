# 🎮 진짜 레벨업: 한별이 키우기

초등학생 한별이의 7일을 대신 살아보며 **"게임을 끊는 법"이 아니라 "게임과 잘 지내는 법"**을 배우는
인터넷·스마트폰 과의존 예방 교육 게임입니다.

- 🍿 팝콘 브레인 지수 · 💪 체력 · 🧠 자기관리 · 😊 행복 — 4개 스탯 균형 잡기
- 🎲 아침 랜덤 이벤트, 🔥 콤보, 🏅 뱃지 10종, 🛍️ 상점
- 😈 "한 판 더?" 다크패턴 체험, 👾 4일차 미드보스, 👹 7일차 최종보스 3라운드
- 🌈 엔딩 4종 (게임도 즐긴 '밸런스 마스터' 히든 엔딩이 최고 등급!)

## 실행

```bash
npm install
npm run dev
```

## 기술

React + Tailwind CSS + Vite. 외부 리소스 없이 이모지/CSS/Web Audio만 사용.
진행 상황은 localStorage에 자동 저장됩니다.
